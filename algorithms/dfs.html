Depth First Search

  • Input:
    • directed or undirected graph with N nodes and M edges
    • source node
  • Output:
    • array reachable of length N, where:
      • reachable[i] = 1 if node i is reachable from source node
      • reachable[i] = 0 otherwise
  • Time complexity:
    • the algorithm works in $O(N + M)$

Description

The idea behind DFS is to go as deep into the graph as possible, and backtrack once you are at a node without any unvisited adjacent node. It is very easy to describe the algorithm recursively.
We start the search at one node. After visiting a node, we further perform a DFS for each adjacent node that we haven't visited before. This way we visit all node that are reachable from the starting node.

Example

Let's see how the Depth First Search algorithm works with an example. We use an undirected graph with 5 nodes.
We start from node 0, the DFS algorithm starts by putting it in the Visited list and putting all its adjacent nodes in the stack.
Next, we visit the node at the top of stack i.e. 1 and go to its adjacent nodes. Since 0 has already been visited, we visit 2 instead.
Node 2 has an unvisited adjacent node in 4, so we add that to the top of the stack and visit it.
Next, we visit the node at the top of stack i.e. 4 and go to its adjacent nodes. Since 4 adjacent nodes already been visited, we visit 3 insted.
After we visit the last node 3, it doesn't have any unvisited adjacent nodes, so we have completed the Depth First Search of the graph.

Code

Python


                        def dfs(
                            graph: Graph,                 # Node-To-Neighbours Dict
                            reachable: Dict[Node, bool],  # Node-Is-Reachable Dict
                            node: Node
                        ):
                            reachable[node] = True
                            for neighbour in graph[node]:
                                if reachable[neighbour] == False:
                                    dfs(graph, reachable, neighbour)
